Steps to write a script

  1. Define the goal: Identify exaxty what is mean to be achieved.
  2. Design the scrips: Split the goal into a series of ordered tasks. A flowchart is a great way to identify and design the steps of the tasks.
  3. Code each task: Write in a programming language the tasks.

About coding...

The code must be writting in a languaje that the computer can undertand, and for this reason we have to use:
