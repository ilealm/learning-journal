Examples of declaring rules for CSS

Each rule has to:

  • After the element name, open the rule " { "
  • After the name attribute, use " : " to indicate the value of the element
  • Separete attributes with " ; "
  • Close element rule with " } "

Examples


body{
font-family:arial;
background-color:beige; }


h1 {
color:darkCyan; }

i {
color:red }
p#pGreen {
color:green; }
