What's make a computer a computer?

Is a device created to help us in the thinking work, to maniputale information, doing 4 task:
  1. Take input: Receive information, instructions, commands from user.
  2. Store information: This are done in the memory of the computer
  3. Process stored information: Take information from memory and procceded by algorits, which are series of commands. This process continuos inteac with the stored information.
  4. Output the results: Gives the user the result of the commands indicated in the algorits.
This devices had evolved since making basic math operations to today technology, but stil have 4 basic elements: Memony, CPU, Input and Output.

How information is represented?
